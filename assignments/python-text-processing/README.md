# 📘 Assignment: Python Text Processing

## 🎯 Objective

Practice essential Python skills for working with strings, reading and writing files, and transforming text data.

## 📝 Tasks

### 🛠️ String Analysis

#### Description
Write a program that analyzes a user-provided sentence and reports the number of words, the number of characters, and the longest word.

#### Requirements
Completed program should:

- Ask the user for a sentence
- Count the total number of words
- Count the total number of characters, including spaces
- Identify and display the longest word
- Example output for "Python is fun to learn" should include:
  - Words: 5
  - Characters: 23
  - Longest word: Python

### 🛠️ File Input and Output

#### Description
Create a script that reads text from a file, processes it, and writes a cleaned version to a new file.

#### Requirements
Completed program should:

- Open an input file and read its contents
- Remove extra whitespace and normalize line breaks
- Write the cleaned text to an output file
- Print a message confirming that the file was saved

### 🛠️ Text Transformation

#### Description
Extend your script to count how many times each word appears and save a summary report.

#### Requirements
Completed program should:

- Split the text into words
- Convert words to lowercase for consistent counting
- Count the frequency of each word
- Save the results to a report file in a readable format
- Example report entry: `python: 2`
